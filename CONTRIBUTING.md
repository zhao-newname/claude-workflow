# Contributing to Claude Workflow

Thank you for your interest in contributing to Claude Workflow!

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Run tests: `npm test`
4. Build: `npm run build`

## Project Structure

- `src/cli/` - CLI commands
- `src/core/` - Core modules (SkillManager, ConfigManager, etc.)
- `src/orchestration/` - Orchestration layer
- `src/agent/` - Agent layer
- `src/types/` - TypeScript type definitions
- `src/utils/` - Utility functions
- `tests/` - Test files

## Coding Standards

- Use TypeScript
- Follow ESLint rules
- Write tests for new features
- Keep functions small and focused
- Document public APIs

## Pull Request Process

1. Create a feature branch
2. Make your changes
3. Add tests
4. Run `npm run lint` and `npm test`
5. Submit a pull request

## Questions?

Open an issue on GitHub.
